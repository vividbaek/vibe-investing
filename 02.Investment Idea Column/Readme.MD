## 권장 읽기 순서

처음 오신 분에게 권장하는 순서:

1. **위험 관리부터** → 1번 (LTCM)
2. **시장 구조 이해** → 4번 (AMC)
3. **섹터 분석** → 6번 (명품) → 13번 (스토리지) → 11번 (AI Supercycle)
4. **정량 분석** → 7번 (BTC-Nasdaq) → 5번 (mNAV) → 3번 (Crypto perp)
5. **AI 시대 투자 리터러시** → 8번 (LLM 합의 vs 현실) → 9번 (Cross-Lingual LLM 학술) → 10번 (LLM Distortion)
6. **재현 가능한 도구** → 12번 (Pharma 프롬프트)

각 자료는 약 5,000–15,000자 분량이며, 핵심 발견 / 다루는 내용 / 데이터 출처 / 알려진 한계 순으로 구성되어 있습니다.

## 칼럼 작성 원칙

이 레포의 칼럼·연구·프롬프트들은 다음 원칙을 따릅니다.

* **실증 데이터 기반** — 인상이나 주관이 아닌 공개 데이터에 근거
* **재현 가능** — 사용한 CSV·프롬프트를 함께 공개
* **법적 안전성** — 시장 구조 분석 시 특정 주체를 지목하지 않고 통계적 귀납만 사용
* **알려진 한계 명시** — 모든 결론에 적용 범위와 한계를 함께 제시
* **다국어 친화** — 영문·중문 readers를 위한 번역 가능성 고려 (Pharma 프롬프트는 KR/EN/CN 동시 제공)
* **1차 출처 우선** — SEC EDGAR, 회사 IR, 공식 학술 출처 우선 인용

> **메타 메시지**: ***\"LLM은 엑셀이지, 정답을 알려주는 오라클(신탁)이 아니다.\"***
