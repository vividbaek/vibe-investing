## ⚙️ Appendix. DPI 모델 계산 공식

### A1. 듀레이션-프리미엄 점수 계산식

```
DPI Score = 0.40 × Sustainability Score + 0.35 × Momentum Score + 0.25 × Valuation Score

Sustainability Score = f(Payout%, FCF Coverage, Debt/EBITDA, Interest Coverage, Buyback)
Momentum Score = w₁·(3Y CAGR) + w₂·(5Y CAGR) + w₃·(10Y CAGR) + w₄·(Payout Stability)
                  where w₁=0.4, w₂=0.35, w₃=0.15, w₄=0.10
Valuation Score = Σ Discount(P/E, P/B, EV/EBITDA vs 5Y avg) + Yield Spread vs 10Y UST
```

### A2. 12개월 기대 총수익률 모델

```
E[Total Return]₁₂ₘ = Dividend Yield + (Forward EPS Growth × Multiple Stability Factor) + Buyback Yield
                    ± Sector Rotation Premium ± Macro Adjustment
```
